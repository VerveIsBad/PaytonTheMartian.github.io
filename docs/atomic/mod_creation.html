Payton Coats / Atomic / Docs # Mod Creation

Documentation for mod creation with Atomic.

There are three required functions in each mod: on_load(basemod), post_load(basemod, mods), and get_name()

on_load(basemod) -> None

This function gets called when your mod is loaded, and because not all mods are yet loaded, you are only given the basemod is provided.

post_load(basemod, mods) -> None

This function gets called after all mods have been loaded, so you can access other mods with the mods argument.

get_name() -> str

This function is called any time something needs to get your mod's name. Whatever you return here is what the name of the mod is.
