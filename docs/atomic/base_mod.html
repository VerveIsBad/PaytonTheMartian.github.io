Payton Coats / Atomic / Docs # Mod Creation

Documentation for mod creation with Atomic.

The Base Mod contains some utilities for modding, and that's pretty much it.

text_mod(mod) -> str

Allows you to modify text, with colors, underlines, etc.

Modifiers:

  • header - Text header
  • blue - Makes text blue
  • cyan - Makes text cyan
  • green - Makes text green
  • warning - Makes the text yellow
  • fail - Makes the text red
  • endc - Reverts text to default
  • bold - Bold.
  • underline - Underline.
